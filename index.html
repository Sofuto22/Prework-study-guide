Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git merge branch: merges the specific branch
  • git add file: add a file to your next commit stage
  • git pull origin main: pulls the base branch to make sure it is up to date
  • git commit -m"add comment": adds a message in with your commit to address changes
